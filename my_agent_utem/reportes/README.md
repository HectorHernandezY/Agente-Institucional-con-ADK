# Carpeta de Reportes

Esta carpeta contiene los reportes PDF generados por la herramienta `generate_pdf_report`.

## Ubicación
Los PDFs se guardan aquí automáticamente cuando se ejecuta la tool `generate_pdf_report`.

## Ruta
`my_agent_utem/reportes/`

## Nota
Los archivos PDF generados están ignorados en Git (ver `.gitignore`).
